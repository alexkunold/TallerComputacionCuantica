**Objetivo del taller**
El objetivo de este taller es introducir la noción de algoritmo cuántico y su implementación en Python mediante Qiskit. Para ello, se construirán circuitos cuánticos con los que se estudiarán fenómenos fundamentales como la superposición y el entrelazamiento. El taller concluirá con la implementación del algoritmo de Deutsch-Jozsa.

15 de abril (Duración: 2 horas)
- 1. Introducción a la computación cuántica (30 minutos)
    - 1.1 ¿Qué motivó la construcción de las computadoras cuánticas?
    - 1.2 ¿Qué computadoras cuánticas existen actualmente?
    - 1.3 ¿Cómo funciona una computadora cuántica?
    - 1.4 ¿Qué problemas pueden resolverse con computadoras cuánticas?

- 2. Compuertas y circuitos cuánticos (1 hora 30 minutos)
    - 2.1 Las compuertas cuánticas como transformaciones unitarias
    - 2.2 El teorema de universalidad
    - 2.3 Instalación de Qiskit en Google Colab
    - 2.4 Superposición cuántica: implementación de un circuito simple
    - 2.5 Entrelazamiento cuántico: uso de la compuerta CNOT

16 de abril (Duración: 2 horas)
- 3. Algoritmos cuánticos
    - 3.1 Funciones de cadenas binarias como transformaciones unitarias
    - 3.2 Funciones constantes y funciones balanceadas
    - 3.3 Análisis teórico del algoritmo de Deutsch-Jozsa
    - 3.4 Implementación del algoritmo de Deutsch-Jozsa en Qiskit
    - 3.5 Pruebas del algoritmo con funciones constantes y balanceadas
